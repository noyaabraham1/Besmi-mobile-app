# Besmi Mobile App

Professional native mobile app for lash booking and business management.

## Features
- Native Android app with authentic Besmi branding
- Push notifications and biometric authentication
- Camera access for photo uploads
- Offline mode with data synchronization
- Smart authentication routing
- Professional onboarding experience

## Build Instructions
1. Go to the "Actions" tab
2. Select "Build Besmi Native Mobile App"
3. Click "Run workflow"
4. Download the APK from the build artifacts
